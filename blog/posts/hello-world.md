这是我的个人网站，也是我的第一篇文章。

## 为什么做这个网站

做这个网站的初衷很简单：

1. **一个线上名片** — 把简历、项目经历、研究成果集中展示
2. **一个技术博客** — 记录水下导航、深度学习、系统工程方面的实践与思考
3. **一个学习项目** — 用纯手工 HTML + CSS + JS 搭建，本身就是一次前端实践

## 这里会写什么

- **技术总结** — UUV 总装总调、湖试/海试中遇到的问题和解决方案
- **论文解读** — 深度学习在水下导航中的应用，包括我自己的研究
- **工具分享** — Python 脚本、QT 开发、嵌入式调试等实用技巧
- **随笔** — 外场试验的故事和感悟

## 关于我

我是赵邵平，西北工业大学航海学院控制工程专业硕士研究生，研究方向为水下航行器导航与控制。

目前最常打交道的东西：UUV、惯性导航、多普勒计程仪、深度学习、Python、QT、Linux……

160+ 天的外场试验经历教会我的最重要的一件事：**在实验室里成立的方案，到湖上不一定成立；但在湖上验证过的方案，一定是靠谱的。**
